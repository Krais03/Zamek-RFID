Architektura systemu:

System składa się z następujących elementów:

Mikrokontroler: Arduino Uno

Czytnik RFID RC522 13,56MHZ

Zamek szafkowy elektromagnetyczny ATLO-DT-C07 12V

Buzzer

Przycisk awaryjnego otwarcia: standardowy przycisk tactile

Backup: zamek mechaniczny

Zasilanie: zasilacz 12V 1A

Przełącznik 5V
